吸血鬼数字的两种解法

发表于 2018-04-09 更新于 2021-02-23 分类于 Algorithm Valine：

吸血鬼数字的解法

基础解法

  根据吸血鬼数字的特点，将a*b=c中的a+b的每一位数字和c中每一位数字排序之后进行比较，如果排序之后的字符串相等，则认为c是一个吸血鬼数字。

  • 代码实现：
    import java.util.Arrays;

public class Vampire {
public static void main(String[] args){
StopWatch stopWatch=new StopWatch();
int counter=0,time=0;
for(int i=10;i<100;i++){
for(int j=i+1;j<100;j++){
int sum=i*j;
if(sum<=1000||sum>9999) continue;
time++;
char[] num1=String.valueOf(i+””+j).toCharArray();
char[] num2=String.valueOf(sum).toCharArray();
Arrays.sort(num1);
Arrays.sort(num2);

            if(Arrays.equals(num1,num2)) {
                counter++;
                System.out.println(i+":"+j+"="+sum+"是吸血鬼数字！");
            }
        }
    }
    System.out.println("总共找到"+counter+"组vampireNumber；\n"+"此程序总共进行了"+time+"次比较，"+"运行时间为："+stopWatch.elapsedTime()+"s");
}

}
//output

21:87=1827是吸血鬼数字！

### 改进算法
在前面的算法中，比较的次数大约为4000次。但是根据下面的方法，可以将比较次数缩减到大约300次：         
&emsp;&emsp;令sum=1000\*a+100\*b+10\*c+d,我们假设他的两个因子(大小肯定为两位数)为：num1=10\*a+b;num2=10\*c+d;那么通过sum-num1-num2我们可以得到sum-num1-num2=990\*a+9\*b=9\*(110a+b).所以sum-num1-num2肯定可以被9整除。  
&emsp;&emsp;在i确定时，我们可以确定j开始循环的范围。所以可以使用start和end来确定j的范围。       
&emsp;&emsp;这告诉我们学好数学的重要性。o_o...

- 代码实现
```java
import java.util.Arrays;

public class VampireNumber {
    public static void main(String[] args){
        StopWatch stopWatch=new StopWatch();
        int counter=0,time=0;
        for(int i=10;i<100;i++){
            int start=Math.max(1000/i,i+1);
            int end=Math.min(10000/i,100);
            for(int j=start;j<end;j++){
                int sum=i*j;
                if(sum%100==0||(sum-i-j)%9!=0)  continue;
                time++;
                char[] num1=String.valueOf(i+""+j).toCharArray();
                char[] num2=String.valueOf(sum).toCharArray();
                Arrays.sort(num1);
                Arrays.sort(num2);

                if(Arrays.equals(num1,num2)) {
                    counter++;
                    System.out.println(i+":"+j+"="+sum+"是吸血鬼数字！");
                }
            }
        }
        System.out.println("总共找到"+counter+"组vampireNumber；\n"+"此程序总共进行了"+time+"次比较，"+"运行时间为："+stopWatch.elapsedTime()+"s");
    }
}

//output

21:60=1260是吸血鬼数字！
21:87=1827是吸血鬼数字！
